# Video Compression Fall 2025 @ NYCU

This repository is dedicated to the programming assignments for the Video Compression course in Fall 2025 at NYCU, lectured by Professor 劉育綸.

## Coursework Overview

The coursework is organized into four homework assignments and a final project, each focusing on different aspects of video compression:

### HW1: Color Transform

- Implementation of color space conversion (e.g., RGB to YCbCr).
- Understanding of how color transform is applied in image/video compression pipelines.

### HW2: 2D-DCT

- Implementation of 2D Discrete Cosine Transform (DCT).
- Exploration of frequency-domain representation of images for compression.

### HW3: Motion Estimation & Compensation

- Implementation of block-based motion estimation algorithms.
- Inter-frame prediction using motion compensation for video compression.

### HW4: Entropy Coding

- Implementation of entropy coding techniques.
- Focus on lossless compression of quantized coefficients.

### Final Project: JPEG Decoder

- Group implementation of a JPEG decoder from scratch.
- Covers the full JPEG decoding pipeline including Huffman decoding, dequantization, IDCT, and color conversion.

## Structure

Each HW directory contains:

- **Source Code**: Python scripts implementing the algorithms.
- **Report**: PDF and DOCX files detailing the methods and results.
- **README**: Instructions and environment requirements for the specific assignment.

The root directory also contains:

- `Final Project JPEG Decoder.pdf` — Oral presentation slides for the final project.
- `VC_Group_33_Final_Project_report.pdf` — Group 33's final project report.
